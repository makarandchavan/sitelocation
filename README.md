# sitelocation
Drupal 9 Module to display sites current location and time.

Installation-
1) Enable this module as you do in Drupal.
2) A block will be placed in the bartik theme on right side region automatically which will display the default timezone.
3) By Default India's country and timezone is configured on installation of this module.
4) You can then set the Country, City and Timezone as per your preference.
5) Time gets refreshed on page refresh after every 60 seconds.
